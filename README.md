## Возможности

- **Готовый Docker образ** - запуск в одну команду
- **Постоянное хранение данных** - пользователи и настройки сохраняются
- **Поддержка VLESS Reality** - современный и безопасный протокол
- **Поддержка доменов и SSL** - Let's Encrypt интеграция
- **Удобный веб-интерфейс** - управление сервером через браузер
- **Статистика и лимиты** - контроль трафика пользователей
- **Автоматический перезапуск** - контейнер всегда в работе
- **Бэкап и восстановление** - простое сохранение данных

## Быстрый старт

### Способ 1: Docker Hub образ

Самый быстрый способ - использовать готовый образ с Docker Hub:

```bash
# Создайте папку для данных
mkdir -p ~/3x-ui/db

# Запустите контейнер
docker run -d \
  --name 3x-ui \
  --restart unless-stopped \
  --network=host \
  -v ~/3x-ui/db:/etc/x-ui \
  knopka123/3x-ui:latest
